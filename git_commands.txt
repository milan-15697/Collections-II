$git init : Initialize the local directory

$git add file_name : Adds file in the local repository and readies them for commit.

$git commit -m "Message" : commits the tracked changes and readies them to be pushed to a remote repo.

$git remote add origin https://github.com/milan-15697/hello-git.git : sets the remote

$git push -u origin master : push the changes in your local repo to remote repo.
